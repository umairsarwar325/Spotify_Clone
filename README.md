# Spotify Clone
This is a frontend web development project created using HTML, CSS, JS.
It includes a builtin folder structure and all songs are fetched from builtin cloud storage

### Features
-> Shows Differnt Folders
-> User can navigate between folders
-> Songs in each folder will be loaded in playlist
-> User can pause, play, next, previous and move time slider

### Live Demo: https://newspotify.freewebhostmost.com/
